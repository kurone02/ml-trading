## Deploying the bot
You can change the stocks of interest in `deploy.py`

```python
TICKERS = [
    "NVDA", "MSFT", "AAPL", "TSLA",
    "META", "PYPL", "GOOG", "COST",
    "INTC", "AMZN", "AMD", "BABA",
]
```

To deploy the bot to Interactive Broker, run the following command

```bash
python deploy.py
```
